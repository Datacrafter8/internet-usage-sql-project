Objective:
Analyze how different age groups use the internet in terms of social media, work, and entertainment.

Tools Used:

SQL (MySQL)

Key Questions Answered:

1-How does social media usage vary by age group?

2-Which age group spends the most time online?

3-What devices are most commonly used by each age group?

4-How does connection type differ across ages?

Key Findings:

  Younger users spend more time on social media.

  Older users spend relatively more time on work/study.

  Mobile is dominant among younger users.

  WiFi usage increases with age.
